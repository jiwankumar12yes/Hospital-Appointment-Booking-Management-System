# Hospital-Appointment-Booking-Management-System
Hospital Appointment Booking Management System built using C# Console Application with N-Layer Architecture, Entity Framework Core, configurable appsettings.json connection string, and integrated Email Notification Service.


🏥 Hospital Appointment Booking Management System

A C# Console Application developed using a clean N-Layer Architecture (UI, BAL, DAL, Shared, Utility) to manage hospital appointments efficiently.

🚀 Key Features
Patient and Doctor Management
Appointment Booking System
Email Notification Service
Entity Framework Core with SQL Server
Configurable database connection via appsettings.json

This project demonstrates structured architecture, separation of concerns, and real-world application design principles.
